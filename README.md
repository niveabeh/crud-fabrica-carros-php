# Sistema CRUD em PHP com MVC

Este projeto foi desenvolvido com foco no meu aprendizado prático e consolidação de conhecimentos em Programação Orientada a Objetos (POO) utilizando a arquitetura MVC (Model-View-Controller).

A aplicação simula um sistema de gerenciamento de veículos, permitindo operações completas de CRUD: Create, Read, Update e Delete, com separação clara de responsabilidades entre as camadas da aplicação.



## Tecnologias Utilizadas

PHP (POO)

Arquitetura MVC

MySQL com phpMyAdmin

PDO (PHP Data Objects)

XAMPP

Bootstrap para interface e responsividade

## Considerações Finais

Este projeto faz parte do meu processo contínuo de aprendizado e evolução na área de desenvolvimento de software, reforçando fundamentos importantes de arquitetura, organização de código e boas práticas em aplicações web.
